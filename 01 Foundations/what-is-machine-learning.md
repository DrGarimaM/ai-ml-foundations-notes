# What is Machine Learning?

Machine Learning (ML) is a branch of Artificial Intelligence that enables systems to learn patterns from data and make decisions or predictions without being explicitly programmed for every rule.

## Simple Intuition

Instead of writing:
"If temperature > 40, turn on AC"

We allow the system to observe temperature data over time and learn patterns automatically.

## Formal Definition

A computer program is said to learn from experience (E) with respect to some task (T) and performance measure (P), if its performance at task T improves with experience E.

## Why Machine Learning Matters

- Handles large-scale data
- Learns patterns humans may not detect
- Enables automation and prediction
- Forms the foundation of AI systems

## Real-World Examples

- Spam email detection
- Recommendation systems
- Face recognition
- Network traffic prediction in telecom systems
